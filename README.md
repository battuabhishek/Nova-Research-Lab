# Personal Portfolio Website

A responsive multi-page portfolio website built with plain HTML, CSS, and JavaScript.

## Pages

- `index.html` - Home page
- `about.html` - About page
- `projects.html` - Projects showcase
- `research.html` - Research highlights
- `contact.html` - Contact page

## Project Structure

```text
.
|-- index.html
|-- about.html
|-- projects.html
|-- research.html
|-- contact.html
|-- styles.css
|-- script.js
`-- assets/
```

## Run Locally

Because this is a static website, you can open `index.html` directly in a browser.

For a better local development experience, use a simple local server:

1. VS Code: install **Live Server** extension
2. Right-click `index.html`
3. Click **Open with Live Server**

## Customize

- Update text content in the HTML files
- Replace images/files inside `assets/`
- Edit styles in `styles.css`
- Update interactive behavior in `script.js`

## Deploy

You can deploy this site for free using:

- **GitHub Pages**
- **Netlify**
- **Vercel**

For GitHub Pages:

1. Push this repository to GitHub
2. Go to **Settings -> Pages**
3. Select branch (usually `main`) and root folder (`/`)
4. Save and wait for the published URL
